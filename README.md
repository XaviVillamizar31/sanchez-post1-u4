# Galería Interactiva

## Descripción

Aplicación web que permite agregar, eliminar y filtrar tarjetas por categoría. También muestra un contador de tarjetas visibles y un mensaje cuando la galería está vacía.

## Instrucciones de ejecución

1. Descargar o clonar el repositorio.
2. Abrir el archivo **index.html** en un navegador web.
3. Usar el formulario para agregar tarjetas y los botones para filtrarlas o eliminarlas.

## Tecnologias Usadas

- HTML5
- CCS3
- JavaScript
